# Realme GT 6T — 3D Model (Three.js)


A realistic interactive 3D model of the Realme GT 6T built with **Three.js (r166)**.  
Fully procedural — no external 3D model files — textures are generated programmatically and environment map is loaded from CDN.

## Features
- Realistic back panel with procedural branding and gradients
- Dual-camera module with lens rings, glass covers and flash
- Mirror & matte finishes, dynamic environment reflections
- Mobile-optimized OrbitControls and responsive GUI (lil-gui)
- Color variants: **Fluid Silver**, **Razor Green**, **Miracle Purple**
- Lightweight and performance-tweaked for mobile (pixel ratio & shadow toggles)
